### Steps:

1. Download and install
   [âž¡ï¸ Download v1.0.0](https://github.com/Hikkywannafly/steam-workshop-download/releases/tag/v1.0.0)
2. Go to **any** Steam Workshop page (e.g., Wallpaper Engine, Left 4 Dead 2,
   Cities: Skylines, etc.).
3. Find a mod, map, or item you want to download and **copy its URL**.\
   Example: `https://steamcommunity.com/sharedfiles/filedetails/?id=3445534475`
4. Open the app and **paste the URL** into the input box.
   - The app will automatically detect the game. If not, you can search for the
     game manually.
5. Set your desired **download folder**.
